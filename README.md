# TaskProject

Employee & Task Management System built with Django.

## 🚀 GitHub Pages Setup

To view the landing page on GitHub Pages:

1. Go to your repository settings
2. Click **Settings** → **Pages** (left sidebar)
3. Under "Build and deployment":
   - Select **Deploy from a branch**
   - Choose branch: **gh-pages**
   - Choose folder: **/ (root)**
4. Click **Save**

Your live page will be available at:
**`https://sandy-rockstar-8998.github.io/Task-Management/`**

## 📋 Project Structure

- **Django Backend**: `TaskProject/` and `TaskApp/`
- **Templates**: `TaskApp/templates/`
- **Static Files**: `TaskApp/static/`
- **Landing Page**: `gh-pages` branch (GitHub Pages)

## 🔧 Deployment Options

GitHub Pages can only serve **static files**. For the full Django app:

- **Render**: https://render.com
- **Railway**: https://railway.app
- **PythonAnywhere**: https://pythonanywhere.com
- **Heroku**: https://heroku.com
