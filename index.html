TaskProject

This repository contains a Django employee/task management app.

GitHub Pages cannot run Django backend code. To deploy the app live, use a Django-capable host such as Render, Railway, or PythonAnywhere.

If you want a static repo landing page, enable GitHub Pages and use:

https://<username>.github.io/<repo-name>/

For the raw Django app, use a deploy platform and your hosted URL will look like:

https://<app-name>.onrender.com
